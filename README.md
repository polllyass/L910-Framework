# Театральная афиша - REST API
##  Описание
REST API для управления театральной афишей и билетами.
##  API для работы с билетами
### Эндпоинты:
| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/tickets` | Получить все билеты |
| GET | `/tickets/:id` | Получить билет по ID |
| POST | `/tickets` | Создать новый билет |
| PUT | `/tickets/:id` | Полностью обновить билет |
| PATCH | `/tickets/:id` | Частично обновить билет |
| DELETE | `/tickets/:id` | Удалить билет |
### Пример билета:
```json
{
  "id": 1,
  "playId": 1,
  "seat": "A12",
  "price": 1500,
  "isSold": false,
  "seatType": "parterre"
}
